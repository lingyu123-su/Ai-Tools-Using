## 📋 活动形式  

活动分为两部分：**产品发布** 和 **打榜**  
1. 🛠️ **产品发布**：在Discussions中发布AI产品并详细介绍功能与使用方法  
2. 🎤 **打榜**：浏览产品，试用后评论，分享体验或推荐类似工具

## 📌 活动流程  

1. **平台设置**  
   - 本仓库启用了Discussions功能，请大家点击顶部菜单的“Discussions”参与讨论。  
2. **产品发布**  
   - 以“[产品发布] 产品名称”为标题，发布你的AI工具推荐帖

     **[产品发布] ChatGPT - 你的万能聊天助手**
